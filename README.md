# Course-Welcome-Page
This is a webpage intended to welcome students of the APP4035 class. It gives them a general idea of the course expectations as well as provides them with the course outline. 

## Tools and Languages
- HTML
- CSS
- Bootstrap
- Node.js
- Handlebars
- Express
